# Automatizacion de evaluacion de candidatos

Skill reutilizable que compara candidatos en PDF con dos vacantes, recomienda el puesto A, el puesto B, ambos o ninguno, y publica una lista priorizada en Notion.

## Contenido

- `SUBMISSION.txt`: enlaces y tiempo de la entrega.
- `agent.md`: instrucciones para ejecutar el flujo.
- `skill/`: codigo completo del skill.
- `NOTION_MCP_SETUP.md`: pasos para conectar Notion.

## Uso rapido

1. Conectar Notion siguiendo `NOTION_MCP_SETUP.md`.
2. Entregar una carpeta con candidatos PDF y dos vacantes PDF.
3. Pedir al agente que use `skill/SKILL.md`.
4. Revisar el reporte publicado antes de tomar decisiones.

Los PDFs de candidatos no se incluyen en el repositorio para proteger sus datos.
